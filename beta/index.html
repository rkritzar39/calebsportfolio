What’s New in v27.0-rc.1

This candidate combines the completed v26.6 beta work with the newest Profile, Live Activity, Discord synchronization, creator, appearance, and cleanup updates.

  • Promoted the v26.6 beta work into the first v27.0 release candidate.
  • Feature-froze v27.0 for final testing and release-blocking fixes.
  • Added the responsive Onyx liquid-glass navigation dock to Home, Release Candidate, Resume, and Settings.
  • Added inline SVG icons, active lens movement, press feedback, drag selection, nearest-item targeting, release snapping, and Settings gear rotation.
  • Improved equal-width navigation, balanced edge spacing, fractional-pixel positioning, safe-area handling, keyboard support, touch behavior, reduced motion, and high contrast.
  • Updated the Profile layout with improved picture-to-name spacing, status presentation, biography readability, Onyx surfaces, and consolidated responsive styling.
  • Updated Live Activity with Spotify progress, PreMiD activities, Discord presence, manual Firestore overrides, dynamic album-art color matching, and accent-color fallback.
  • Added optional Discord synchronization for display name, profile picture, and online status, including preview, refresh, fallback, and saved synchronization settings.
  • Redesigned TikTok, Instagram, and YouTube creator cards.
  • Updated public-facing device and website information without exposing sensitive device details.
  • Updated the Resume page to share the Settings appearance system and improved light, dark, clear, tinted, print, and browser PDF layouts.
  • Removed retired interfaces, obsolete code, unused listeners, and unnecessary public processing.
