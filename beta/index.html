What’s New in v26.6-beta.4

This release introduces the responsive Onyx navigation dock and improves the Resume page across appearance modes, screen sizes, and input methods.

  • Added the responsive Onyx liquid-glass navigation dock to the Home, Beta, Resume, and Settings pages.
  • Added inline SVG navigation icons, an active liquid lens, press feedback, drag selection, and release snapping.
  • Added a Settings-only gear animation that replays when the Settings destination is selected.
  • Improved dock alignment with equal-width navigation items, balanced edge spacing, and fractional-pixel lens positioning.
  • Added responsive dock layouts for phones, foldables, tablets, laptops, desktops, landscape displays, and split-screen layouts.
  • Improved keyboard navigation, reduced-motion support, high-contrast support, safe-area handling, and touch-device behavior.
  • Updated the Resume page to use the shared appearance system from the Settings page.
  • Improved Resume page support for light mode, dark mode, clear mode, tinted mode, printing, and browser-based PDF layouts.

Beta Notes

Important information about this beta version.

  • This beta is not the final September revamp.
  • Some visual styles and interactions may change before release.
  • Liquid-glass effects remain experimental.
  • Phone, foldable, tablet, landscape, desktop, and split-screen layouts may still be adjusted.
  • The Onyx navigation dock may receive additional animation, spacing, and compatibility refinements.
  • Custom backgrounds may affect contrast and readability.
  • The main creator shoutout sections remain active.
  • The final complete revamp is planned as v27.0.
