What’s New in v26.6-beta.3

This release focuses on removing retired features and reducing unnecessary website code.

  • Removed the FAQ, Posts, Blog, President information, Legislation tracker, Quote of the Day, Latest TikTok embed, Onyx AI assistant, and Project Goal Tracker.
  • Removed associated JavaScript, CSS, Firestore references, realtime listeners, page routing, and initialization calls.
  • Reduced unnecessary public-page processing and unused data activity.
  • Kept the main TikTok, Instagram, and YouTube creator shoutout sections.
  • Kept device AI compatibility information in the Tech Information section because it is separate from the retired Onyx AI assistant.
