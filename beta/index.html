What’s New in v27.0 RC 2

This second candidate adds the advanced Education portfolio, five-item Onyx navigation, README automation, and platform-planning updates while retaining the Profile, Live Activity, Discord synchronization, creator, appearance, and cleanup work from the first release candidate.

  • Added the advanced Education portfolio and public Education page.
  • Added academic profile, degree progress, requirement groups, semesters, courses, grading modes, GPA and credit calculations, timelines, goals, planned coursework, achievements, certifications, projects, skills, and transfer-credit tools.
  • Added Education publishing, privacy, JSON backup, import, print-summary, search, filtering, and last-updated controls.
  • Expanded the responsive Onyx liquid-glass navigation dock to Home, Release Candidate, Education, Resume, and Settings.
  • Added Education-specific responsive navigation for phones, foldables, tablets, laptops, desktops, short landscape screens, keyboard input, reduced motion, and safe areas.
  • Added workflow-compatible README metadata automation and corrected AUTO marker support.
  • Updated the future platform direction to a link-in-bio and social platform called Caleb's Social Talk.
  • Removed the job-board, recruiting, applicant-tracking, and candidate-pipeline direction from the future roadmap.
  • Promoted the v26.6 beta work into the first v27.0 release candidate.
  • Feature-froze v27.0 for final testing and release-blocking fixes.
  • Added the responsive Onyx liquid-glass navigation dock, later expanded in RC 2 to include Education.
  • Added inline SVG icons, active lens movement, press feedback, drag selection, nearest-item targeting, release snapping, and Settings gear rotation.
  • Improved equal-width navigation, balanced edge spacing, fractional-pixel positioning, safe-area handling, keyboard support, touch behavior, reduced motion, and high contrast.
  • Updated the Profile layout with improved picture-to-name spacing, status presentation, biography readability, Onyx surfaces, and consolidated responsive styling.
  • Updated Live Activity with Spotify progress, PreMiD activities, Discord presence, manual Firestore overrides, dynamic album-art color matching, and accent-color fallback.
  • Added optional Discord synchronization for display name, profile picture, and online status, including preview, refresh, fallback, and saved synchronization settings.
  • Redesigned TikTok, Instagram, and YouTube creator cards.
  • Updated public-facing device and website information without exposing sensitive device details.
  • Updated the Resume page to share the Settings appearance system and improved light, dark, clear, tinted, print, and browser PDF layouts.
  • Removed retired interfaces, obsolete code, unused listeners, and unnecessary public processing.

Release Candidate Notes

Important information about v27.0 RC 2.

  • v26.5 remains the current stable public version until v27.0 is formally released.
  • v27.0 RC 2 is the public display name; v27.0-rc.2 is the technical identifier and recommended Git tag.
  • v27.0 RC 2 is feature-frozen and intended for final validation.
  • Only release-blocking bugs, accessibility corrections, compatibility fixes, performance issues, and content corrections should be added.
  • Liquid-glass effects and custom backgrounds may vary by browser and device.
  • The main creator shoutout sections remain active.
  • The final v27.0 release is targeted for late July or early August 2026.
